# Java Console-Based Project Collection (OOP + Fundamentals)

This repository contains multiple Java mini-projects built to practice and demonstrate Core Java concepts such as:
- **OOP Principles** (Encapsulation, Inheritance, Polymorphism, Abstraction)
- **Control Statements** (loops, conditional statements)
- **Collections & Arrays**
- **Class, Objects, Constructors, Methods**
- **Problem-solving using Java**

Each project runs in the console and focuses on real-time use cases to simulate how Java is used in day-to-day applications.



##  Projects Included

| Project Name | Description | Concepts Used |
|--------------|-------------|---------------|
| **Atm-Simulator** | Simulates ATM operations like PIN verification, balance check, withdrawal, and deposit. | Classes, Objects, Encapsulation, Conditional statements |
| **Bank-Account-Management-System** | Creates and manages multiple customer accounts, allowing deposit/withdraw/transfer. | Encapsulation, Constructors, Arrays |
| **Employee-Leave-System** | Tracks employee leave balance, leave requests, and approval workflow. Handles different leave types. | Encapsulation, Inheritance, Polymorphism |
| **Employee-Payroll-System** | Calculates salary based on attendance, tax deductions, and allowances. | Classes, Objects, Methods |
| **Hotel-Booking-System** | Books rooms based on availability and displays booking summary. | OOP, Arrays, Encapsulation |
| **Library-Management-System** | Borrow/return books and maintain borrowing history. | ArrayList, Encapsulation |
| **Online-Shopping-Cart** | Add items to cart, calculate total bill, and generate invoice. | Classes, Polymorphism |
| **Quiz-Application** | Console-based quiz app with scoring logic. Supports multiple categories. | Encapsulation, Polymorphism (method overloading) |
| **StudentGradeSystem** | Stores student marks and calculates grades with result summary. | Arrays, Encapsulation |
| **Vehicle-Management-System** | Manages vehicle details and calculates rental cost based on type. | Inheritance, Method overriding |
