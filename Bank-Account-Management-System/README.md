# Bank Account Management System (Java)

## Project Overview
The **Bank Account Management System** is a console-based Java application that allows users to create accounts, deposit money, withdraw money, and check balance.  
This project was built to practice Core Java concepts and apply **Object-Oriented Programming (OOP)** in a practical scenario.

## Features
- Create a new bank account
- Deposit and withdraw money (with validation)
- Check current balance
- Menu-driven console interface for user interaction

## Concepts Used
| Concept | Usage |
|---------|--------|
| Encapsulation | Account details stored as private fields |
| Classes & Objects | `BankAccount` and `BankSystem` classes |
| Methods | `deposit()`, `withdraw()`, `checkBalance()` |
| Conditional Statements | Validating withdrawals and user choices |
